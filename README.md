# Desktop AI Mascot

デスクトップに常駐し、あなたの作業をサポートするAIエージェントです。
マルチモーダル対応（画像解析）、長期記憶、スキル実行、そしてプラグインによる拡張機能を備えています。

## 特徴

- **マルチモーダル解析**: 画像やファイルをドラッグ＆ドロップするだけで、AIが内容を理解・要約します。
- **長期記憶 (Vector Search)**: 過去の会話を意味ベースで記憶し、数日前の話も踏まえた回答が可能です。
- **プラグインシステム**: `plugins/` フォルダにPythonファイルを置くだけで、誰でも新機能を追加できます。
- **マルチモデル対応**: Gemini (Google) と Groq (Llama) を自由に切り替えて利用可能。
- **実用スキル**: リマインダー設定、ファイル検索、WEB検索などのスキルを標準搭載。

## インストール

1. Python 3.10以上をインストールしてください。
2. 依存ライブラリをインストールします：
   ```bash
   pip install -r requirements.txt
   ```

## 使い方

### Windows
`DesktopAI.pyw` をダブルクリックして起動してください。（コンソールは表示されません）

### Linux / Mac
```bash
python3 main.py
```

起動後、キャラクターを右クリックして「設定」からAPIキーを入力してください。

## プラグインの作り方
`plugins/` フォルダに `BaseSkill` を継承したクラスを作成することで、簡単に機能拡張が可能です。詳細は `skills/` 内のサンプルを参照してください。
